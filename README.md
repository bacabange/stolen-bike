# Stolen Bike

Challenge for Chipper

## Tech Requirements

* Typescript
* Code Linter (ESlint)
* Tailwindcss
* styled-components (for dumb components)
* Testing: jest, react-testing-library

## Available Scripts

In the project directory, you can run:

* `yarn start` http://localhost:3000
* `yarn test`
* `yarn build`
* `yarn lint`
